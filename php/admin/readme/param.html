PHP传值给FLASH

玩家进入步骤

  1. 玩家点击 4399 里面的游戏链接进入 youyuegame 游戏网页 http://gameServer/index.php
  2. 服务器端index.php 接受到请求，查询用户是否存在，并且生成JS数据传送给SWF文件
  3. FLASH游戏文件接受JS数据并且加载
  4. 进入游戏成功，正式开始

◆ 传值参数列表及说明

Key 说明与要求
4399uid 当前用户的4399的账号ID
userid 当前用户在 youyuegame 的账号ID
timestamp 进入本游戏页面时候的时间戳
username 当前用户的用户名
serverid 当前服务器的服号，其值格式S1,S2,S3
currurl

当前URL
